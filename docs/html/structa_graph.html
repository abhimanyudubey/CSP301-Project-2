vector<string> aGraph::nodeList

Basic graph structure for Analyzer. nodeList : is the list of all node labels. sourceList : is the list of sources for each edge. targetList : is the list of targets for each edge. edgeList : in graphML each edge has a label (which may be optional), and we store the label of each edge in this list. adjacencyMatrix : this variable is the adjacencyMatrix of the graph, or a matrix of boolean variables where adjacencyMatrix(i,j) = 1 if i and j are connected, and adjacencyMatrix(i,j) = 0 if they are not.

In our case, we use a non-directed graph, so the adjacencyMatrix is symmetric.
