Thread generate\-Courses creates courses “once every semester.\-” It generates courses and registers students. In other words it edits students' course data. It does not create any new nodes. It also registers students in a course – each student is registered for “semester\-\_\-dept\-\_\-courses” courses of their department and “semester\-\_\-nondept\-\_\-courses” courses of some other (randomly chosen) departments.
